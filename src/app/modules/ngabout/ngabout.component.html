Angular

Платформа для разработки веб-приложений, написанная на языке TypeScript, разрабатываемая командой из компании Google

Angular api и RxJs имеют довольно большое Api и функционал, но для реализации продукта, в данном случае UiKit, можно использовать только необходимое API. Ниже собраны все инструменты, которые пригодятся для реализации компонент. Рекомендую их повторить, если забыли или вы не успели их попробовать.
