Subject
Это класс помеченный декоратором Directive. С помощью нее можно добавлять новое поведение существующему DOM элементу.
Основные черты: в одном DOM элементе может быть много директив. У директив нет View (template). Внутри директивы нельзя использовать пайпы.
Когда использовать: создание масок, dragAndDrop, button и тд.
BehaviorSubject
Это класс помеченный декоратором Component. Созданы для того, чтобы разбивать приложение на маленькие кусочки.
Основные черты: в одном DOM элементе может быть 1 компонент, в отличие от директив. Есть View (template). Внутри компонента используются пайпы.
Когда использовать: создание представления.
Subscription
Это класс реализующий interface PipeTransform. Нужны для преобразования данных при их выводе в HTML.
Основные черты: Реализует метод transform(value: any, ...args: any[]) и возвращает преобразованное значение.
Когда использовать: курсы валют, форматирование чисел и дат, преобразование Enum в текст.
Частая ошибка - использовать функцию в шаблоне там где нужен Pipe.
