Angular API

NgContent, NgTemplate, NgContainer, NgTemplateOutlet
Позволяют создавать динамичные и настраиваемые компоненты.
NgContent позволяет родительским компонентам внедрять код html в дочерние компоненты
NgTemplate: директив-шаблон, контент которой может быть отображен в другом месте.
NgContainer: позволяет применить структурную директиву к разделу страницы, не создавай при этом дополнительный элемент
NgTemplateOutlet: может взять NgTemplate шаблон и и создать его экземпляр где угодно на странице.
ViewChild, ElementRef, TemplateRef, ViewContainerRef
ViewChild: получает доступ в DOM элементу
Основные черты: в одном DOM элементе может быть 1 компонент, в отличие от директив. Есть View (template). Внутри компонента используются пайпы.
Когда использовать: создание представления.
ControlValueAccessor
Это класс реализующий interface PipeTransform. Нужны для преобразования данных при их выводе в HTML.
Основные черты: Реализует метод transform(value: any, ...args: any[]) и возвращает преобразованное значение.
Когда использовать: курсы валют, форматирование чисел и дат, преобразование Enum в текст.
Частая ошибка - использовать функцию в шаблоне там где нужен Pipe.
